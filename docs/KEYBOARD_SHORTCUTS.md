## Pro Tips

### Efficiency Tips

1. **Two-Hand Technique:**
   - Left hand: Arrow keys, Space, H, U
   - Right hand: 1-9, Delete, Tab
   - Faster than one-handed typing

2. **Minimize Hand Movement:**
   - Keep hands in home position
   - Use shortcuts that are close together
   - Avoid reaching for mouse

3. **Develop Muscle Memory:**
   - Practice same shortcuts repeatedly
   - Use keyboard-only mode (hide mouse)
   - Within 1-2 weeks, shortcuts become automatic

4. **Combine Shortcuts:**
   - Arrow â†’ Number â†’ Tab (place and move)
   - Space â†’ 1, 3, 5 â†’ Space (quick candidates)
   - H â†’ Arrow â†’ Number (hint and continue)

5. **Use Tab Aggressively:**
   - Jump between empty cells
   - Faster than arrow keys for distant cells
   - Perfect for filling naked singles

### Troubleshooting

**Shortcut Not Working?**

1. **Check browser focus:**
   - Click on puzzle area first
   - Some shortcuts only work when puzzle is focused

2. **Browser conflicts:**
   - Ctrl+S may trigger "Save page"
   - Disable browser extension shortcuts
   - Try incognito mode

3. **Keyboard layout:**
   - Some layouts (AZERTY, Dvorak) may differ
   - Number row should work universally
   - Adjust to your layout

4. **Operating system shortcuts:**
   - Mac may intercept some Cmd+ shortcuts
   - Windows may intercept some Ctrl+ shortcuts
   - Use alternative shortcuts if available

**Still Issues?**
- Check [Help Center](HELP_CENTER.md)
- Report to support with browser/OS details
- Use mouse as fallback
