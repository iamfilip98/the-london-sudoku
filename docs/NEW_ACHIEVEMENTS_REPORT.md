## Design Philosophy

### Progressive Difficulty
- **Common (26)**: First steps, initial milestones (1-10 completions, basic participation)
- **Rare (42)**: Moderate commitment (10-50 completions, 7-30 day streaks, league participation)
- **Epic (32)**: Significant dedication (50-100 completions, 30-180 day streaks, competitive success)
- **Legendary (10)**: Elite accomplishments (1000+ completions, 365+ day streaks, perfect seasons)

### Engagement Drivers
1. **Immediate Gratification**: 26 common achievements for quick wins
2. **Medium-Term Goals**: 42 rare achievements for ongoing engagement
3. **Long-Term Commitment**: 32 epic achievements for dedicated players
4. **Ultimate Challenges**: 10 legendary achievements for completionists

### Player Retention
- **Daily Play**: Streak achievements encourage daily logins
- **Variety**: Variant achievements promote exploration of all game modes
- **Social**: Friend and league achievements drive multiplayer engagement
- **Competition**: League achievements create competitive incentives
- **Improvement**: Skill achievements reward personal growth

## Implementation Requirements

### Database Schema
```sql
-- Existing tables (no changes needed)
achievements (id, name, description, category, rarity, xp, icon, requirement_type, requirement_value)
user_achievements (user_id, achievement_id, earned_at, progress)

-- New tracking fields needed in user_stats
ALTER TABLE user_stats ADD COLUMN IF NOT EXISTS variant_completions JSONB DEFAULT '{}';
ALTER TABLE user_stats ADD COLUMN IF NOT EXISTS perfect_streaks JSONB DEFAULT '{}';
ALTER TABLE user_stats ADD COLUMN IF NOT EXISTS error_free_streak INT DEFAULT 0;
ALTER TABLE user_stats ADD COLUMN IF NOT EXISTS personal_best_streak INT DEFAULT 0;
```

### API Endpoints (Existing)
- `GET /api/achievements` - Fetch available achievements
- `POST /api/achievements/unlock` - Unlock achievement
- `GET /api/stats?type=achievements` - Get user progress

### Tracking Requirements

#### Per Achievement Type:
1. **Variant Completions**: Count per variant per user
2. **Perfect Streaks**: Current streak counter per variant
3. **League Performance**: Season history, finish positions, points
4. **Friend Challenges**: Win/loss records
5. **Daily Streaks**: Last play date, current streak
6. **Battle Pass**: XP per season, tier reached, rewards claimed
7. **Error-Free Streak**: Current error-free game count
8. **Personal Bests**: Time history per difficulty

#### Event Triggers:
- On puzzle completion ‚Üí Check variant, perfect, streak achievements
- On league season end ‚Üí Check league achievements
- On friend challenge complete ‚Üí Check social achievements
- On daily login ‚Üí Check streak achievements
- On Battle Pass XP gain ‚Üí Check BP achievements
